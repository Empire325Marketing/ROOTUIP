Overview

ROOTUIP is an enterprise-grade container tracking and demurrage prevention platform that helps shipping companies save millions in detention and demurrage fees. Our platform integrates with all major shipping carriers and provides real-time visibility into your container movements.

Key Features

  • Real-time Tracking: Monitor containers across 50+ carriers with unified visibility
  • D&D Prevention: AI-powered predictions prevent up to 90% of demurrage charges
  • Multi-source Integration: Connect via APIs, web scraping, EDI, and email parsing
  • Advanced Analytics: Comprehensive reporting and predictive insights
  • Enterprise Security: SOC 2 compliant with end-to-end encryption

Quick Start Guide

Get up and running with ROOTUIP in just 5 minutes. Follow these steps to start tracking your first container:

1. Get Your API Key

First, sign up for a ROOTUIP account and generate your API key from the dashboard:

bash 
# Set your API key as an environment variable
export ROOTUIP_API_KEY="your_api_key_here"

# Or add to your .env file
echo "ROOTUIP_API_KEY=your_api_key_here" >> .env

2. Install the SDK

Choose your preferred programming language and install the ROOTUIP SDK:

bash 
npm install @rootuip/sdk

3. Initialize the Client

Create a new instance of the ROOTUIP client with your API key:

javascript 
const { RootuipClient } = require('@rootuip/sdk');

// Initialize the client
const client = new RootuipClient({
    apiKey: process.env.ROOTUIP_API_KEY,
    environment: 'production' // or 'sandbox' for testing
});

// Test the connection
const status = await client.getStatus();
console.log('Connected to ROOTUIP:', status);

4. Track Your First Container

Now you can start tracking containers with a simple API call:

POST /api/v3/containers/track
javascript 
// Track a container
const container = await client.containers.track({
    containerNumber: 'MSKU1234567',
    carrier: 'maersk',
    referenceNumber: 'BOOK123456'
});

console.log('Container status:', container.status);
console.log('Current location:', container.currentLocation);
console.log('ETA:', container.eta);

// Set up real-time updates
client.containers.subscribe(container.id, (update) => {
    console.log('Container update:', update);
});

API Reference

Authentication

All API requests must include your API key in the Authorization header:

http 
Authorization: Bearer your_api_key_here
Content-Type: application/json

Container Tracking Endpoint

POST /api/v3/containers/track

Start tracking a container by providing the container number and carrier information.

Request Parameters

Parameter Type Required Description
containerNumber string Required The container identification number
carrier string Required Carrier code (e.g., 'maersk', 'msc', 'cma-cgm')
referenceNumber string Booking or B/L reference number
webhookUrl string URL to receive real-time updates

Example Request

bash 
curl -X POST https://api.rootuip.com/v3/containers/track \
  -H "Authorization: Bearer your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "containerNumber": "MSKU1234567",
    "carrier": "maersk",
    "referenceNumber": "BOOK123456",
    "webhookUrl": "https://your-domain.com/webhooks/container-updates"
  }'
