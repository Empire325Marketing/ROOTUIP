Usage Guidelines

  • Clear Space: Maintain minimum clear space equal to the height of the "U" in ROOTUIP
  • Minimum Size: Logo should never be displayed smaller than 120px wide (primary) or 24px (icon)
  • Background: Use on white or light backgrounds. Use inverted version on dark backgrounds
  • Value Proposition: When space permits, include "$500K per vessel" messaging
  • Typography: Always use Arial or system fonts for consistency
  • Colors: Never alter the brand colors. Use only approved color combinations
