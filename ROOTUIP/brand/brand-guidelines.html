Logo Usage Rules

✓ DO

  • Maintain clear space equal to logo height
  • Use primary logo on light backgrounds
  • Scale proportionally
  • Use monochrome on dark backgrounds
  • Respect minimum sizes

✗ DON'T

  • Stretch or distort logo
  • Use on busy backgrounds
  • Change colors or gradients
  • Add effects or shadows
  • Use below minimum sizes

Voice & Tone Guidelines

Our voice reflects enterprise expertise while remaining approachable. We speak with authority about complex logistics challenges.

✓ Our Voice

  • Authoritative: We know ocean freight
  • Results-focused: Measurable ROI
  • Technical: Precise terminology
  • Confident: Proven solutions
  • Professional: Enterprise-grade
Example:

"Stop losing $14M per vessel to preventable D&D charges. Our AI platform integrates with ALL carrier systems to eliminate inefficiencies in 48 hours."

✗ Not Our Voice

  • Casual: Avoid slang or jokes
  • Vague: No generic promises
  • Timid: Don't hedge claims
  • Jargon-heavy: Stay accessible
  • Overselling: Let results speak
Avoid:

"We might be able to help you save some money on shipping costs with our cool new platform."

Key Messaging

Primary Value Prop: "Stop Losing $14M Per Vessel to Ocean Freight Inefficiencies"
Differentiation: "The ONLY platform that connects ALL carrier systems"
Proof Point: "94% D&D reduction in 48 hours"
Target Audience: "Enterprise importers with 10+ vessels"

Implementation Guidelines

Technical specifications for developers and designers implementing the UIP brand system.

CSS Implementation

/* Include brand system files */
@import 'brand/enterprise-colors.css';
@import 'brand/enterprise-typography.css';

/* Use CSS custom properties */
color: var(--uip-primary-600);
font-family: var(--uip-font-primary);
font-size: var(--uip-text-lg);

Asset Requirements

  • Logos: SVG format for scalability, PNG fallbacks for legacy support
  • Icons: 16px, 24px, 32px, 48px, 64px sizes minimum
  • Images: WebP with JPEG fallbacks, optimized for performance
  • Favicons: ICO, PNG, SVG formats for cross-browser compatibility

Performance Standards

  • Load Time: <2 seconds first contentful paint
  • Font Loading: System font fallbacks, font-display: swap
  • Color Contrast: WCAG AA compliant (4.5:1 minimum)
  • Responsive: Mobile-first, tested on all devices
